# DEPRICATED

## Install

```javascript
    npm install
    npm start // Starts the devserver
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

- Redux 4.0.0
- Redux thunk 2.2.0

- Redux-Devtools-Extension

### Dev dependencies
- Babel 6.26.0
- Babel polyfill
- Webpack 4
- Webpack dev server
- CSS/Style/SASS loaders
- Node-Sass

... and more
